      💱 匯率設計說明

      三欄定義

      • 參考匯率：從台銀牌告 API 抓取的市場即時匯率，唯讀，僅供參考。
      • 固定加成：系統預設百分比（建議 3%），自動計算在參考匯率上。
      • 報價匯率：實際計算用，業務可手動覆蓋。旅行業慣例比市場匯率高 2–5%。

      旅行業匯率慣例

      機票款採航空公司每月公告匯率；地面費用採台銀即期匯率加 2–4%。包套報價建議鎖匯（固定加成），避免出發前匯率波動造成虧損。

      📣 Banner 管理規格

      • 建議尺寸：1200 × 400 px（比例 3:1）
      • 檔案格式：JPG / PNG / WebP，上限 2 MB
      • 最多顯示：5 張，自動輪播間隔預設 5 秒（3–30 秒可調）
      • 權限：只有管理員可新增 / 編輯 / 刪除，一般業務只能瀏覽
